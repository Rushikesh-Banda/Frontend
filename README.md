# Important Frontend Concepts

---

# HTML5

## Definition

HTML (HyperText Markup Language) is used to create the structure of web pages.

### Common Elements

| Tag        | Purpose   |
| ---------- | --------- |
| `<div>`    | Container |
| `<h1>`     | Heading   |
| `<p>`      | Paragraph |
| `<img>`    | Image     |
| `<button>` | Button    |

---

# CSS3

## Definition

CSS (Cascading Style Sheets) is used to style HTML elements.

### Features

* Colors
* Layouts
* Animations
* Responsive design

---

# Responsive Web Design

## Definition

Responsive design makes webpages adapt to different screen sizes.

### Benefits

* Mobile-friendly UI
* Better user experience
* Flexible layouts

---

# Flexbox

## Definition

Flexbox is a CSS layout system used for aligning items.

### Example

```css id="g1m5n7"
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

---

# Media Queries

## Definition

Media queries apply different styles for different screen sizes.

### Example

```css id="r7t3m1"
@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}
```
